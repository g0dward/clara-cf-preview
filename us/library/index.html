Buttons

Use Button.astro — variants primary / secondary / white / ghost, sizes sm / md / lg / xl.

Variants

Sizes

Button group · responsive stack

When two CTAs share a row, on mobile they should stack vertically and stretch to the full container width — otherwise they look ragged and the secondary CTA reads as an afterthought. Wrap them in w-full flex flex-col sm:flex-row items-stretch sm:items-center sm:justify-center and pass fullWidth with class="sm:w-auto" on each button.

Resize the window: buttons stretch to container width on mobile, sit inline ≥640px.

<div class="w-full flex flex-col sm:flex-row items-stretch
            sm:items-center sm:justify-center gap-3">
  <Button fullWidth class="sm:w-auto">Start free trial</Button>
  <Button variant="secondary" fullWidth class="sm:w-auto">How it works</Button>
</div>

Shadows

Clara uses layered shadows (multiple stacked drops) for premium softness. Always favour the layered recipe over a single big shadow.
